# MCBEReTw
適用於Minecraft基岩版的繁體中文譯名修正資源包。

大家,是否覺得基岩版的物品名跟Java版的物品名稱不太一樣呢?

這並不是錯覺,是真的在亂翻譯（#

因為聽說是官方請了翻譯團隊來進行翻譯而翻出了一堆奇妙的翻譯,不像Java版可以進行投票翻譯

所以我就做了把基岩版的物品名統一成Java版的一個資源包

原翻譯：獄髓護腿->獄髓緊身褲?

投擲器->墜落遊戲??

扭曲蕈菇釣竿->怪奇蕈菇釣竿???

地獄疙瘩->地獄節結????

避雷針->雷霆法杖?????
